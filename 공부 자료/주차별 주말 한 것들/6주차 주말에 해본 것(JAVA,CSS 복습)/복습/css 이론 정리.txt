CSS(Cascading Style Sheets)
InLine Sheet
각 태그에 직접 style 속성을 지정하여 적용하는 방법
Internal Sheet
head 부분에 style 태그를 만들어서 특정 태그들에 style 속성을 지정하는 방법
External Sheet
(link 태그,@import)를 이용하여 외부에서 style 속성 값들을 불러와서 적용시키는 방법

선택자 
*1전체 선택자
*{속성: 값;....}

*2태그 선택자
태그명{속성: 값;...}

*3클래스 선택자(기본적으로 앞에 아무것도 안붙이면 root태그인 html태그로 설정된다.)
태그.클래스 명{속성:값;...}



*4ID 선택자
#id속성 값{속성: 값;....}




*5복합 선택자

1.하위 선택자(자손 선택자)

상위태그명 하위 태그명{...}


2.자식 선택자

상위태그명>하위 태그명{...}

인접 형제 선택자
(부모 태그명)+(자식 태그명){..}
일반 형제 선택자
(부모 태그명)~(자식 태그명){..}


*6속성 선택자

속성이 포함된 요소를 선택
[attribute※" "]

※?
(없음) 해당 속성이 포함된 요소를 선택
~= 오른쪽 피연산자의 값이 포함된 값을 포함한 요소를 선택
^= 오른쪽 피연산자의 값으로 시작하는 요소를 선택
$=
