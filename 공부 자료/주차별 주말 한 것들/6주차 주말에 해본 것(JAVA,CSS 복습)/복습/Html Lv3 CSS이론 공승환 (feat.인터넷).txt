CSS(Cascading Style Sheets)
html문서 내에 미디어 요소가 어떻게 보이는지 정의하는 스타일 언어
선언방식

Internal Sheet
html 파일에 태그 안에 내용을 입력해 스타일링하는 방식. 따로 파일을 만들지 않아도 된다는 장점이 있지만, html과 css 각각의 코드가 길어질 경우 작업효율이 떨어지고 가급적 모든 파일을 따로 분류해 만드는것을 권장한다.
External Sheet
link태그를 사용해 외부 css파일을 연결하는 방식.
Inline Sheet
요소의 style 속성에 직접 스타일을 작성하는 방식으로 선택자가 없다. 유지보수에 악영향을 미칠수 있으므로 권장하진 않음.
@import 방식 css의 임포트규칙으로 css문서 안에서 또다른 css문서를 가져와 연결하는 방식

Inline /Block 요소
- 블럭요소 : 하나의 태그가 브라우저에서 좌우공간을 
다 차지하면서 독립적인 덩어리 공간을 가지는 요소들입니다.
- 인라인요소 : 하나의 태그가 브라우저에서 실제로 코딩된 
그 영역만 차지하여 좌우로 다른 태그가 나란히 위치할 수 있는 요소들입니다. 
행 안의 일부분이라고 보시면 됩니다. 텍스트레벨요소라고도 합니다.
둘 다 body내에서 단독으로 사용가능

선택자
기본
* : 전체 선택자. 모든 요소를 선택할 때 사용한다.
태그선택자 : 태그이름을 선택할 때 사용. ex: div, span…
클래스선택자 : .classname 와같이 클래스속성의 값을 선택한다.
아이디선택자: #idname 와같이 아이디속성의 값을 선택한다.

복합
태그.클래스네임 : 선택자 두개를 함께 지칭해 동시만족하는 요소선택 ex: span.classname
선택자 > 자식선택자 : 선택자의 자식요소를 선택하는 방식. ex: ul > .orange
선택자 하위선택자 : 선택자의 하위요소를 선택하는 방식. ex: div .orange
선택자 + 선택자 : 선택자의 형제요소를 선택하는 방식. 선택자의 인접한곳에 위치한 형제를 찾는다. 즉 선택자의 다음 요소. ex: .bro + li
선택자 ~ 선택자 : 선택자의 다음형제 요소 모두를 선택 ex: .orange ~ li

가상클래스
선택자:hover : 선택자 요소에 마우스커서가 올라가있는동안 지정해둔 css가 실행된다.
선택자:active : 선택자 요소에 마우스를 클릭한 동안 지정해둔 css가 실행된다.
선택자:focus : 선택자 요소가 포커스된동안 지정해둔 css가 실행된다.
선택자:link : 선택자 요소가 방문하지 않은 링크상태 시 적용
선택자:visited : 선택자 요소가 방문한 링크 상태 시 적용
선택자:first-child : 선택자가 형제요소 중에 첫번째인것을 선택.
선택자:last-child : 선택자가 형제요소 중에 막내인것을 선택.
선택자:nth-child(n) : 선택자가 형제요소 중에 (n)번째 인것을 선택.
선택자:not(제외선택자) : 제외선택자를 제외한 요소를 선택.

가상요소
선택자::before : 선택자 요소의 내부 앞에 content를 삽입.
선택자::after : 선택자 요소의 내부 뒤에 content를 삽입.
선택자::first-letter : 선택자 텍스트의 첫 글자만 적용
선택자::first-line : 선택자 텍스트의 첫 줄만 적용
선택자::selection : 사용자가 드래그한 텍스트만 적용

속성
[선택자] : 속성 선택자를 포함한 요소를 선택
[선택자=“value”] : 속성값이 value에 해당하는 요소를 선택.


기타 가상 클래스
:target 기존 선택자 양식 뒤에 붙이는 형식으로 클릭 후에 
:target뒤에 붙는 중괄호 style을 적용시키는 가상 클래스

:lang 인용구 끝 부분에 특수기호 추가 시에 사용하는 가상클래스
:in-range   input 태그에 입력된 속성 값이 범위 내에 있을 경우
:out-range    input 태그에 입력된 속성 값이 범위 바깥에 있을 경우
:read-only readonly 속성을 포함한 input 요소에 적용
:read-write 읽기 쓰기 모두 가능한 태그 적용(default)
:required required 속성을 가지고 있는 태그들에게 적용
:optional required 속성을 가지지 않은 태그들에게 적용
:valid 유효한 값을 가진 태그에게 적용
:invalid 유효하지 않은 값을 가지는 태그에게 적용
:not(선택자) 파라미터로 온 선택자 이외의 모든 태그에게 적용





psuedo element
::first-letter 텍스트 중 제일 첫 글자에게 효과 부여시 사용하는 가상 요소
::first-line 텍스트 중 제일 첫 줄에게 효과 부여시 사용하는 가상요소
::before 단락이 시작 바로 전에 적용하는 의사 요소
::after 단락이 끝난 바로 뒤에 적용하는 의사 요소
::selection 사용자가 드래그 할 시 효과가 나타나는 의사 요소
