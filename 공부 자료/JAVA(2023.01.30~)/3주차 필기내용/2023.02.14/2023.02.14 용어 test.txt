OS >>운영체제 사용자가 컴퓨터를 보다 편리하게 사용하기 위해 만들어진 소프트웨어

OSS(OpenSourceSoftware): 누구나 사용할 수 있도록 범용화된 소프트웨어

범용 소프트웨어: 이윤 추구목적이 아닌 누구나 사용할 수 있는 소프트웨어

상용 소프트웨어: 이윤 추구목적으로 만들어진 소프트웨어

IDE(Integrated Development Environment):통합 개발 환경

SDK(Software Development Kit): 소프트웨어 개발 도구

블랙박스 테스트: 모듈의 내부구조를 비공개로 테스트하는 방식

화이트박스 테스트:모듈의 내부구조를 공개로하여 세부적으로 테스트하는 방식

Driver: 소프트웨어 간에 통신을 위해 필요한 소프트웨어
(상향식 통합 테스트에서 더미 모듈에 해당)
Stub: 하향식 통합 테스트에서 종속되는 모듈을 대체하는 모듈을 뜻한다.

네트워크 공격 방식

Bruth Force Attack: 무차별 대입 공격 

Land Attack: 대상 ip에  ack를 버퍼가 오버플로우 현상이 일어나도록 보내서 해당 ip에 대한 서비스를 중지 시키는 공격

IP Spoofing: IP 또는 DNS주소를 위조하여 사기를 치는 행위..(**보완해서 외우기!)
+ 위조 html문서를 보내 클라이언트 컴퓨터 권한을 뺏거나 개인정보를 빼내는 행위

IP Sniffing : 해당 ip가 패킷을 교환 시에 해당 패킷을 훔쳐보는 행위

Watering Hole : 특정 ip가 자주 접속하는 사이트를 조사하여 해당 사이트에 악성코드를 심어서
해당 ip에 악성코드를 심는 공격

Zero Day Attack: 아직 알려지지 않은 해킹방식으로 공격하는 행위.

ZombiePC(bot):악성코드에 감염된 pc를 뜻한다. 다른 말로 bot(봇)이라고 한다.

Bot Net: Bot으로만 이루어진 네트워크 망을 뜻한다.

+)
Smurfing:희생자의 스푸핑된 원본 IP를 가진 수많은 인터넷 제어 메시지 프로토콜 패킷들이 
IP 브로드캐스트 주소를 사용하여 컴퓨터 네트워크로 브로드캐스트하는 분산 서비스 거부 공격

ICMP: 인터넷 프로토콜의 비신뢰적인 특성을 보완하기 위한 프로토콜(네트워크 분야)

Hash Code: 객체를 식별하는 하나의 정수값(암호화하는 하나의 방법이라고 생각하면 편하다.)
(주소값과 다른 값이며, 저장공간은 인덱스,주소값, 해시 코드 전부 다른 공간에 저장)
