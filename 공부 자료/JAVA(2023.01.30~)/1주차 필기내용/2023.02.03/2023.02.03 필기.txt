즉 기본형을 제외한 변수를 뜻한다.

printf(format,arguments)
format은 서식문자가 포함된 String 값, arguments 는 서식문자에 대응하는 변수나 리터럴 값.





equals 함수와 비교연산자 ==의 차이

equals 함수는 저장된 리터럴 값을 비교하는데, ==연산자는 객체가 같은지 비교하기에 예외상황 발생 가능성이 존재
고로 문자열 비교 시에 equals 함수를 쓰는게 좋다.

equals 함수는 Object 클래스의 메소드인가?  아니면 String 클래스의 메소드인가?


Object 클래스의 equals 함수는 본래 객체의 주소가 같은지 비교하는 메소드이다.
Answer
String 클래스에서 오버로딩한 메소드 (Object 클래스는 모든 클래스의 부모클래스 즉 최상위 클래스이다.)

매개변수 타입차이로 오버로딩한 것으로 추정(매개변수 개수는 한 개)

String 클래스의 equals함수의 매개변수는 String , Object 클래스의 equals함수의 매개변수 타입은 Object

상수 vs 리터럴
상수는 개발자가 1번은 값을 주소에 저장할 수 있으나 리터럴은 개발자가 뭔 짓거리를 해도 바꿀 수 없다.
리터럴은 상수이다.
상수: 값을 저장할 수 있는 공간
리터럴:값을 한 번 저장하면 변경할 수 없는 저장공간

Big O 표기법(알고리즘 시간 복잡도)

 O(1) <O(log n) < O(n) <O(nlogn) < O(n^2)<O(2^n)
코드의 처리속도를 보는 척도라고 보면 된다.

 for문이 n번 돈다고 본다(컴퓨터 연산횟수) 그러면 이중 for문은 n^2이고
시간 복잡도는 O(n),O(n^2)이다
