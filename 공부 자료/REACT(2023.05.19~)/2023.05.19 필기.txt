용어 정리

JSX
JS에서 확장된 문법의 언어
React에서는 본질적으로 렌더링 로직이
 UI 로직(이벤트가 처리되는 방식, 
시간에 따라 state가 변하는 방식, 
화면에 표시하기 위해 데이터가 준비되는
방식 등)과 연결된다는 사실을 받아들임
 
NPM(Node Package Manager)
npm은 Node.js로 만들어진 모듈을 웹에서 받아서 설치하고 
관리해주는 프로그램
npx는 패키지를 한번 실행한다는 의미

Bundler
기능별로 모듈화 했던 파일들을 하나로 묶어주는 작업

React의 State
리액트에서 State는 Component의 데이터 상태를 말한다. 
한마디로 State란 개발자가 정의한 변경 가능한 데이터를 말함.

React의 Props
React가 사용자 정의 컴포넌트로 작성한
엘리먼트를 발견하면 JSX 어트리뷰트와 
자식을 해당 컴포넌트에 전달되는 객체

Babel
최신 문법을 써도 이전 버전 브라우저 환경에서도 
사용할 수 있도록 최신 문법을 특정 예전버전의 js로 
변환해주는 툴(짧게 '이전문법으로 바꿔주는 API'로 숙지)

웹훅
 비즈니스 로직을 호스팅하거나 다른 서비스를 호출하는 서비스
