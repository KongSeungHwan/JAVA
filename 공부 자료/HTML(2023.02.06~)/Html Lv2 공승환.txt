Html이란?

HyperTextMarkupLanguage

하이퍼텍스트마크업 언어


클라이언트 측에서 요구한 정보를 서버 측에서 html이라는 문서를 통해 데이터를 보여주는 파일(HTTP 기반)

HTTP란? 
HyperTextTransferProtocol

html같은 문서를 구성하는 언어의 규약이라고 설명할 수 있다.(통신규약)



실제 사이트에서 보였던 이상한 꺾쇠 <> <> 이런거 알고보니 tag라는 명칭이 있음

시작<> 종료</>(시작~종료 범위에 있는 것들은 해당 태그의 자식 관계이다.)


JSP의 구조

지시어 태그
 jsp,html을 만들게 되면 최상단에 위치한 태그
<%@ ~~>

속성=값 이렇게 쌍으로 속성을 지정할 수 있는 구조다.
다중으로도 가능!
<%@ page language="java" contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"%> >>데이터를 보낼때 헤더에 해당하는 부분(네트워크 OSI 7계층때 배움)
<!DOCTYPE html> >>html의 버전을 웹브라우저에 알리는 태그
<meta> >> meta Data를 정의할때 사용
<html> >> 웹페이지의 시작과 끝을 나타내는 태그
<head> >> 문서의 머리를 나타내는 태그
<body> >> html문서의 텍스트,하이퍼링크,이미지,리스트 등과 같은 모든 컨텐츠를 포함하는 태그
기본으로 jsp파일을 만들게 되면 이렇게 나타난다.

특정 속성


다중 속성 정의 예시
<p style ="color:red; background-color:blue; text-align:center"></p>
p 태그 안의 글자색을 빨강색으로 바꾸고 해당 영역을 파랑색으로 바꾸고 글 정렬을 중앙으로 한다는 뜻
3가지 속성을 정의

style 속성
	color 값 해당 영역의 글자색을 바꾸는 효과를 줌
	background-color 값 해당 영역의 배경화면 색을 바꾸는 효과를 줌
font-family 속성
	couier 값 해당 영역의 글씨체를 얇게 하는 효과를 줌
	verdina 값 해당 영역의 글씨체를 굵게 하는 효과를 줌
font-size 속성
	값 단위 px(절대값), %(상대값)

text-align 속성
center,left,right 등등

<a>태그 속성 href
	값 url 쓰는 속성

<div>태그(많이 씀)
margin 요소 간격
padding 경계를 밀어내는 수치
border 선의 두께
<p>태그 내부에서 다시 또 영역을 나누는 태그

<span> 해당 영역의 글자에 해당하는 영역만 효과를 적용하는 태그(많이 씀)

다룰 영역을 지정하는 태그(유용)

<img>태그
src 속성값 해당 이미지의 url 속성
alt 속성값 이미지를 불러올 수 없을 때 예외 처리(문자열로 쳐야함)

<map> 태그
map과 usemap의 변수로 이어진 태그의 영역을 다시 영역을 나눈 것.
해당 속성 name 값 usemap 속성과 이어진 이름을 적는 곳
<area> 태그 map태그와 같이 쓰는 태그인데 이는 영역을 원,사각형,디폴트(전체영역),다각형
<ul>,<li>
ul은 리스트의 전체 영역을 뜻하는 태그,li는 리스트 중 한 행에 해당하는 영역을 나타내는 태그
<table>,<tr>,<th>,<td>
table 표 전체의 영역을 뜻하는 태그,tr은 표의 한 행의 영역을 다루는 태그,th는 맨 윗 행 영역의 요소를 나열할 수 있는 태그
td는 th영역을 제외한 나머지 tr영역

<button>
버튼을 만드는 태그
<form>
서버 페이지에 매개변수를 같이 넘겨주는 태그
<submit>
form 태그 내에 존재하는 태그로, 매개변수를 넘겨줌과 동시에 페이지이동이 이루어진다. 
<iframe>
웹페이지안에 또 다른 웹페이지를 만드는 태그
