OS(Operating System): 운영체제, 사용자가 컴퓨터를 보다 편리하게 사용하기 위한 소프트웨어
(예전에는 cmd창으로 파일시스템으로 관리를 했으나 현재는 GUI로 사용하여 파일 시스템을 관리하는 게 예시)


상용 <-> 범용 소프트웨어
이윤            이윤x


OpenSource(Open Source Software, OSS):
오픈소스 소프트웨어는 공개적으로 액세스할 수 있게 설계되어 누구나 자유롭게 확인, 수정, 배포할 수 있는 코드

License:  개인이나 다른 기업에서 먼저 개발한 사용권을 허가를 통해 사용하는 계약

Driver: 드라이버(driver)는 컴퓨터에 설치된 하드웨어를 사용할 수 있게 해주는 프로그램

Memory:시스템의 단기 데이터 스토리지

우리가 기본적으로 알고 있는 RAM(Random Access Memory)을 이야기하는것
통칭 램은 프로그램을 임의의 위치에 할당하여 실행하는 메모리

컴파일러: 우리가 작성한 코드를 기계어로 바꿔주는 일종의 번역기임.

인터프리터 VS 컴파일러
동작 방식 차이임.

Integrated Development Environment
통칭 IDE 통합 개발 환경을 말함.

SDK(Software Development Kit): 소프트웨어 개발 도구 모음
