line style
dotted >> 점 선
dashed >> 선을 띄엄띄엄 자른 듯한 형태
solid>> 단일 선
double >> 이중 선
groove >> 이중 선이나 바깥 선은 짙은 색, 안쪽은 원본 색 굵기 수치가 double 보다 좀 크다
ridge>> groove와 반대로 바깥 선이 원본 색 안쪽 선이 짙은 색
inset >> 굵은 선 테두리이며 사선으로 잘라 두영역으로 나눠 위쪽 영역은 짙은 색, 아래쪽은 원본 색이다.
outset >> inset과 반대 색상이며 굵은 선 테두리이다.
none >> 효과적용x
hidden >> 선을 숨기는 효과
